# Устанока:
```
git clone https://github.com/GeorgeVorob/Laravel-workers-list.git
```
```
cd Laravel-workers-list/
```
```
npm install
```
```
composer install
```
```
php artisan migrate
```
```
php artisan storage:link
```
```
npm run dev
```
Во втором окне терминала:
```
php artisan serve
```
